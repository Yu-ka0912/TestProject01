Pleiades All in One は開発対象となる言語によりパッケージを選択できる日本語 Eclipse ディストリビュー
ションです。インストールはダウンロードした zip ファイルを解凍するだけです。

ンスの記述を参照してください。

・初期設定デフォルトフォントの Windows 8.1、10 対応追加
・PHP 版に Pleiades 初回起動時のオプションを指定
　init.batch=../xampp/setup_xampp_no_pause.bat

4.5.2.v20160228

・Pleiades、各種プラグイン、ランタイムバージョンアップ
・Tomcat Launcher を Tomcat 8 対応、機能追加されたフォーク版に変更
・php.ini に date.timezone = Asia/Tokyo を設定
・C/C++ 64bit 版に MSYS を入れ忘れていたため追加 (rm コマンドが無いためプロジェクトクリーンでエラー)

・自動デフォルト設定追加
　一般 > ワークスペース > ウィンドウ・タイトルにワークスペース・パスを表示 : ON
　パッケージ・エクスプローラー・ビュー > パッケージ･プレゼンテーション : 階層
　Maven > ユーザー・インターフェース > デフォルトの POM エディターで XML ページを開く : ON
　F11 で常に前回起動したアプリケーションを起動する
　C/C++ &gt; 新規 C/C++ プロジェクト・ウィザード : MinGW GCC を優先ツールチェーンにする

4.5.1.v20151002

・Pleiades、各種プラグイン、ランタイムバージョンアップ
・Minimalist Gradle Editor 追加 (Buildship にエディタが含まれるようになるまで)
・WTP Tomcat の SetPropertiesRule 警告ログを抑止

・自動デフォルト設定追加
　Web > JSP ファイル ： エンコード = ISO 10646/Unicode(UTF-8)
　一般 > コンテンツ・タイプ ： JSP (配下すべて) > デフォルト・エンコード = UTF-8

・自動デフォルト設定変更
　一般 > Glance 検索 : 始動時に表示 = ON → OFF、クローズボタンの表示 = ON → OFF
　　Glance 検索フィールドを表示していると Git ヒストリー・ビューや Gradle ビューの表示などが
　　くずれるため、上記の初期設定を変更。表示 Ctrl + J、クローズはフォーカスがある時に ESC。
